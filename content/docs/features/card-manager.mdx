## 功能概述

卡片管理器允许你管理背屏中常驻显示的卡片。每张卡片对应一个目标应用的组件，你可以自定义卡片的标题、目标包名、组件名称、优先级以及启用状态。

<Callout>
  卡片配置修改后，建议重启 com.xiaomi.subscreencenter 以使更改生效。
</Callout>

---

## 功能说明

### 卡片属性

每张卡片包含以下可配置属性：

| 属性 | 说明 |
| --- | --- |
| **标题** | 卡片在背屏中显示的名称 |
| **目标包名** | 卡片所指向的应用包名（如 `com.android.music`） |
| **组件名称** | 应用中具体组件的类名 |
| **优先级** | 卡片的排列顺序，数值越小排在越前面 |
| **启用状态** | 控制卡片是否在背屏中显示 |

## 使用方法

### 查看现有卡片

1. 打开 **REAREye** 应用
2. 进入 **背屏中心** → **卡片管理器**
3. 查看当前已配置的所有卡片列表

### 添加新卡片

1. 在卡片管理器中点击添加按钮
2. 填写卡片信息：
   - 输入 **标题**（用于标记该卡片的用途）
   - 输入 **目标包名**（应用的 package name 默认即可）
   - 输入 **组件名称**（卡片所对应的组件名称 - 需要在组件管理器中存在）
   - 设置 **优先级**（决定显示顺序）
3. 确认保存
4. 确保卡片的启用状态为开启

### 编辑卡片

1. 在卡片管理器中点击要编辑的卡片
2. 修改需要变更的属性
3. 保存更改

### 删除卡片

1. 在卡片管理器中找到要删除的卡片
2. 选择删除并确认

### 调整优先级

通过修改卡片的优先级数值来调整其在背屏中的排列顺序：

- 数值越**小**，卡片排在越**前面**
- 数值越**大**，卡片排在越**后面**

### 启用 / 禁用卡片

- **启用**：卡片将在背屏中正常显示
- **禁用**：卡片保留在配置中但不显示，方便后续重新启用

---

## 使用场景

- **快捷启动**：添加常用应用的卡片，方便在背屏快速打开
- **信息展示**：配置信息类组件的卡片，在背屏实时查看
- **优先级管理**：将重要的卡片排在前面，减少翻找

---

## 注意事项

- 目标包名和组件名称需要填写准确，否则卡片可能无法正常工作
- 修改卡片配置后，建议重启 `com.xiaomi.subscreencenter`
- 部分应用的组件可能未适配背屏显示，效果取决于应用本身
- 可以通过禁用而非删除的方式临时隐藏卡片

<Callout type="warn">
  修改目标包名或组件名称时请确认填写正确。错误的配置可能导致卡片无法显示或显示异常内容。
</Callout>
