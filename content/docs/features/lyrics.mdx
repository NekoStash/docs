## 功能概述

REAREye 的歌词系统**不直接提供**背屏歌词显示功能，而是通过整合歌词数据源，让小米主题原有的歌词功能在背屏上复活。

### 工作原理

```
词幕插件 (Lyricon) / SuperLyric  →  获取歌词数据  →  REAREye 转接  →  主题组件显示歌词
```

1. **歌词提供器**（Lyricon 插件 / SuperLyric）负责从音乐应用获取歌词
2. **REAREye** 作为中间层，支持切换和管理不同的歌词提供器
3. **主题组件**（如背屏音乐主题）负责在背屏上实际渲染和显示歌词

<Callout type="warn">
  仅安装 REAREye 无法直接在背屏显示歌词。你还需要安装支持歌词显示的主题组件。
</Callout>

## 前置条件

使用歌词功能前，请确保：

1. 已安装并激活 **REAREye** 模块（参考 [安装指南](/docs/basic/installation)）
2. 已安装支持歌词显示的**背屏主题组件**（见下方说明）
3. 已安装**歌词插件**（Lyricon 词幕插件或 SuperLyric）

## 歌词提供器

REAREye 通过歌词提供器获取歌词数据。目前支持两种提供器。

### Lyricon（词幕）

Lyricon 是一个模块化的歌词系统，通过**插件**从不同的音乐应用获取歌词。REAREye 对接的是 Lyricon 的插件层。

#### 安装

你只需要安装**词幕插件**，无需安装 Lyricon 主体：

1. 从 [LyricProvider Releases](https://github.com/tomakino/LyricProvider/releases) 下载对应音乐应用的插件
2. 在设备上安装插件 APK
3. 在 **LSPosed** 管理器中启用插件模块
4. 确认作用域包含目标音乐应用
5. 重启设备或相关应用

<Callout>
  Lyricon 插件按音乐应用分类，如网易云音乐、QQ 音乐等。根据你使用的音乐应用下载对应插件即可。
</Callout>

<Callout type="warn">
  如果您需要同时使用 REAREye 和 Lyricon 请前往QQ群下载修改版
</Callout>

### SuperLyric

SuperLyric 是另一个歌词提供器，同样为 REAREye 提供歌词数据。

#### 安装

1. 获取 SuperLyric 的 APK 安装包（可通过 QQ 群获取）
2. 在设备上安装 APK
3. 在 **LSPosed** 管理器中启用 SuperLyric 模块
4. 确认作用域配置正确
5. 重启设备或相关应用

<Callout>
  Lyricon 插件和 SuperLyric 可以同时安装，之后在 REAREye 设置中按需切换使用。
</Callout>

## 音乐控件白名单

在使用歌词功能之前，需要将你的音乐应用添加到音乐控件白名单中。

### 设置步骤

1. 打开 **REAREye** 应用
2. 进入 **背屏中心** → **音乐控件白名单**
3. 勾选你使用的音乐应用（如 QQ 音乐、网易云音乐等）
4. 修改后建议重启 `com.xiaomi.subscreencenter`

### 强制刷新音乐控件状态

部分音乐应用在切歌或播放状态变化后，背屏控件可能不会及时更新。你可以开启「强制刷新音乐控件状态」来解决此问题。

## 歌词显示模式

当歌词系统正常工作后，可以在 REAREye 中配置显示模式：

### 原文

显示歌曲的原始歌词文本。

### 翻译

如果歌词提供器包含翻译数据，可显示歌词的翻译版本。

### 罗马音

显示歌词的罗马音标注（适用于日语、韩语等歌曲）。

### 设置方法

1. 打开 **REAREye** 应用
2. 进入 **背屏中心** → **歌词显示模式**
3. 选择你偏好的显示模式

---

## 切换歌词提供器

如果你同时安装了 Lyricon 插件和 SuperLyric，可以在 REAREye 中切换使用。

1. 打开 **REAREye** 应用
2. 进入 **背屏中心** → **歌词提供器**
3. 选择 **Lyricon** 或 **SuperLyric**

<Callout>
  不同的提供器可能支持不同的歌曲和歌词格式。如果某个提供器找不到歌词，可以尝试切换到另一个。
</Callout>

---

## 逐行歌词显示

REAREye 支持逐行歌词显示模式，可以更精确地跟随播放进度。

1. 打开 **REAREye** 应用
2. 进入 **背屏中心** → **逐行歌词显示模式**
3. 选择显示 **原文** 或 **翻译**

## 常见问题

### 背屏上没有显示歌词？

请检查以下几点：

1. 是否安装了支持歌词显示的**主题组件**
2. 是否安装了**歌词插件**（Lyricon 插件或 SuperLyric）
3. 音乐应用是否已添加到**音乐控件白名单**
4. LSPosed 中 REAREye 作用域是否包含 `com.xiaomi.subscreencenter`
5. 尝试重启 `com.xiaomi.subscreencenter` 或重启设备

### 歌词不同步或显示不完整？

1. 尝试开启「强制刷新音乐控件状态」
2. 尝试切换不同的歌词提供器
3. 部分歌曲可能暂无歌词数据，属于正常现象

### Lyricon 插件和 Lyricon 主体有什么区别？

- **Lyricon（词幕）主体**：完整的状态栏歌词应用，独立于 REAREye 使用
- **Lyricon 词幕插件**：仅为 REAREye 提供歌词数据，无需安装主体

使用 REAREye 的歌词功能时，**只需安装插件**即可。
